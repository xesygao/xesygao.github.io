Spring Cloud Finchley - Eureka配置详解

配置详解

原始配置文件说明在spring-cloud-netflix-eureka-client-2.0.0.RELEASE.jar!/META-INF/spring-configuration-metadata.json

eureka.instance配置

参数 说明 备注
eureka.instance.appname 注册到eureka的服务名 默认值unknown
eureka.instance.appGroupName 注册到eureka的服务组名 默认无
eureka.instance.instanceEnabledOnit 是否使实例一注册到eureka服务端就开始分担流量 默认false
eureka.instance.nonSecurePort http端口 默认80
eureka.instance.securePort https端口 默认443
eureka.instance.nonSecurePortEnabled 是否启用http 默认true
eureka.instance.securePortEnabled 是否启用https 默认false
eureka.instance.leaseRenewalIntervalInSeconds 心跳间隔时间 默认30(s)
eureka.instance.leaseExpirationDurationInSeconds 服务中心删除此服务实例的等待时间(秒为单位),时间间隔为最后一次服务中心接受到的心跳时间与当前时间的时间差 默认90(s)
eureka.instance.virtualHostName 该实例的虚拟主机名称(http) 默认值”unknown”
eureka.instance.instanceId 实例id 默认无，在同一个appName下该实例id唯一
eureka.instance.secureVirtualHostName 该实例的虚拟主机名称(https) 默认值”unknown”
eureka.instance.aSGName
eureka.instance.metadataMap 自定义的元数据 默认无，配置例子:eureka.instance.metadataMap.dataname=datavalue
eureka.instance.dataCenterInfo 该实例所在数据中心信息 默认无
eureka.instance.ipAddress 该实例ip地址 默认无
eureka.instance.preferIpAddress 是否优先使用ip地址 默认false
eureka.instance.statusPageUrlPath 该实例的状态呈现url,相对路径 默认值”\status”
eureka.instance.statusPageUrl 该实例的状态呈现url,绝对路径 默认无
eureka.instance.homePageUrlPath 该实例的主页url,相对路径 默认”/“
eureka.instance.homePageUrl 该实例的主页url,绝对路径 默认无
eureka.instance.healthCheckUrlPath 该实例健康检查url,相对路径 默认”/health”
eureka.instance.healthCheckUrl 该实例健康检查url,绝对路径 默认无
eureka.instance.secureHealthCheckUrl 该实例的安全健康检查url,绝对路径
eureka.instance.namespace 该服务实例的命名空间,用于查找属性 默认值”eureka”,在springcloud中被忽略
eureka.instance.hostname 注册主机名 默认无
eureka.instance.initialStatus 注册到eureka的初始状态 默认无,例如:(UP、DOWN、STARTING、OUT_OF_SERVICE、UNKNOW)
eureka.instance.defaultAddressResolutionOrder 默认的地址解析顺序 默认值”[]”
eureka.instance.environment 该实例的环境配置 默认无,如果不配置，则为test

eureka.client配置

参数 说明 备注
eureka.client.propertyResolver
eureka.client.enabled 是否启用eureka客户端 默认值true
eureka.client.transport
eureka.client.registryFetchIntervalSeconds 从eureka服务器注册表中获取注册信息的时间间隔(s) 默认为30秒
eureka.client.instanceInfoReplicationIntervalSeconds 复制实例变化信息到eureka服务器的时间间隔(s) 默认为30秒
eureka.client.initialInstanceInfoReplicationIntervalSeconds 最初复制实例信息到eureka服务器所需的时间间隔(s) 默认为40秒
eureka.client.eurekaServiceUrlPollIntervalSeconds 询问Eurekaservice url信息变化的时间间隔(s) 默认为300(s)
eureka.client.proxyHost eureka服务的代理主机 默认无
eureka.client.proxyPort eureka服务的代理端口 默认无
eureka.client.proxyUserName eureka服务的代理用户名 默认无
eureka.client.proxyPassword eureka服务的代理密码 默认无
eureka.client.eurekaServerReadTimeoutSeconds eureka服务读取超时前等待的时间 默认为8秒
eureka.client.eurekaServerConnectTimeoutSeconds eureka服务连接超时前等待的时间 默认为5秒
eureka.client.backupRegistryImpl 实现了BackupRegistry接口的类名 该类作为eureka客户端第一次获取注册信息时的失败回退选项
eureka.client.eurekaServerTotalConnections eureka客户端可连接eureka服务端连接总数 默认是200
eureka.client.eurekaServerTotalConnectionsPerHost eureka客户端可连接eureka服务端主机总数 默认是50
eureka.client.eurekaServerURLContext eureka注册中心路径 默认无。该选项在eureka.client.useDnsForFetchingServiceUrls值为true时需配置，用于构造service url
eureka.client.eurekaServerPort eureka服务端口 该选项在eureka.client.useDnsForFetchingServiceUrls值为true时需配置，用于构造service url
eureka.client.eurekaServerDNSName dns名 默认值无，该选项在eureka.client.useDnsForFetchingServiceUrls值为true时需配置
eureka.client.region 当前实例所属region 默认值”us-east-1”
eureka.client.eurekaConnectionIdleTimeoutSeconds 到eureka服务端的http连接关闭前允许闲置时间 默认值30(s)
eureka.client.registryRefreshSingleVipAddress
eureka.client.heartbeatExecutorThreadPoolSize 初始心跳执行器(heartbeatExecutor)线程池大小 默认值2
eureka.client.heartbeatExecutorExponentialBackOffBound 心跳执行器指数退避策略相关属性，其值为重试延迟的最大倍数 默认为10
eureka.client.cacheRefreshExecutorThreadPoolSize 初始缓存刷新执行器(cacheRefreshExecutor)线程池大小 默认值2
eureka.client.cacheRefreshExecutorExponentialBackOffBound 缓存刷新执行器指数退避策略相关属性，其值为重试延迟的最大倍数 默认为10。获取注册信息的操作也是一个异步任务，每隔30秒执行一次（通过eureka.client.registryFetchIntervalSeconds配置），如果操作失败，也是以2的指数形式延长重试时间，直到达到eureka.client.registryFetchIntervalSeconds * eureka.client.cacheRefreshExecutorExponentialBackOffBound 这个上限，之后一直以这个上限值作为重试间隔，直至重新获取到注册信息，并且重新尝试获取注册信息的次数是不受限制的。
eureka.client.serviceUrl 指定服务注册中心地址 如果服务注册中心为高可用集群时，多个注册中心地址以逗号分隔
eureka.client.gZipContent 是否开启gzip压缩 默认true，可以减少网络流量
eureka.client.useDnsForFetchingServiceUrls 是否通过dns获取eureka服务端列表 默认值false
eureka.client.registerWithEureka 是否注册到eureka服务端 默认true
eureka.client.preferSameZoneEureka 优先选择同一机房实例 默认true
eureka.client.logDeltaDiff 是否记录eureka客户端与服务端之间注册信息的差异 默认false
eureka.client.disableDelta 是否禁用增量更新从而获取全量注册信息 默认值为false
eureka.client.fetchRemoteRegionsRegistry 一串由”,”分隔的rigion列表，eureka客户端从这些rigion获取注册信息 配置了该项则需要配置eureka.client.availabilityZones
eureka.client.availabilityZones 当前实例所属rigion可用zone列表 rigion和zone是AWS中的概念，rigion相当于一个区域，zone相当于一个机房
eureka.client.filterOnlyUpInstances 是否只获取状态为”UP”的实例 默认true
eureka.client.fetchRegistry 是否从eureka服务端获取注册信息 默认true
eureka.client.dollarReplacement 在eureka服务端序列化/反序列化信息时替换$的字符串 默认值为_-
eureka.client.escapeCharReplacement 在eureka服务端序列化/反序列化信息时替换_的字符串 默认值__
eureka.client.allowRedirects 是否允许将请求重定向到另一个服务/集群 默认值false
eureka.client.onDemandUpdateStatusChange 如果设置为true，则通过ApplicationInfoManager进行的本地状态更新将触发对远程eureka服务器的按需(但限速)注册/更新 默认true
eureka.client.encoderName 编码器名称 临时配置，当最新编解码器稳定时该配置项将被移除
eureka.client.decoderName 解码器名称 临时配置，当最新编解码器稳定时该配置项将被移除
eureka.client.clientDataAccept EurekaAccept name for client data accept
eureka.client.shouldUnregisterOnShutdown 是否在关闭时取消注册 默认true
eureka.client.shouldEnforceRegistrationAtInit 是否在初始化阶段强行注册到eureka server 默认false
eureka.client.order
